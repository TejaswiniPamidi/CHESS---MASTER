# CHESS MASTER ♟️

## Description
Chess Master is a Java-based chess game developed as part of my academic project.
The application follows standard chess rules and demonstrates object-oriented design.

## Project Structure
- `com.chess.engine` – Core game logic and main entry point
- `com.chess.gui` – Graphical user interface components

## Features
- Standard chess board initialization
- Turn-based gameplay
- Modular and clean code structure
- GUI-based interaction

## Technologies Used
- Language: Java
- Concepts: OOP (Encapsulation, Inheritance, Polymorphism)
- IDE: IntelliJ IDEA

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `Chessv2.java` (main class) from `com.chess.engine`
